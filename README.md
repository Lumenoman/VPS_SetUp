VPS_SetUp
Basic server setup
Универсальный скрипт базовой настройки VPS v2.9.3.3
Поддерживаемые ОС: Debian/Ubuntu
Авторы в порядке вклада: ChatGPT, Grok, DeepSeek, Lumenoman

Сохрани скрипт на сервере:
mkdir /usr/local/bin/VPS_SetUp/
nano /usr/local/bin/VPS_SetUp/VPS_SetUp.sh
Сделай исполняемым:
chmod +x /usr/local/bin/VPS_SetUp/VPS_SetUp.sh
Запусти: /usr/local/bin/VPS_SetUp/VPS_SetUp.sh
Скрипт логирует всё в /usr/local/bin/VPS_SetUp/setup.log
И сохраняет параметры в /usr/local/bin/VPS_SetUp/server_env.txt

Или можно установить скрипт командой:
```bash
curl -fsSL https://raw.githubusercontent.com/Lumenoman/VPS_SetUp/main/install.sh | bash
```

Возможности.
Запись логов ведется в файл setup.log
Сбор важных для администрирования параметров сервера и запись их в файл server_env.txt
Изменение пароля root на 32х символьный, включающий латинские буквы в разных регистрах, цифры и символы.
Смена порта SSH (NPort).
Добавление дополнительного порта (XPort) для сервисов которые могут быть установлены позже.
Обновление системы.
Установка часто используемы пакетов.
Ужесточение конфигурации SSH (вход только по ключу, ограничение попыток и т.д.).
Создание ключей.
Создание готовой команды подключения к серверу.
Активация и конфигурация защиты от брутфорса (Fail2Ban для порта NPort).
Активация и конфигурация фаервола (UFW).
Включение BBR и оптимизация параметров sysctl.
Отключение IPv6.
Смена часового пояса.
Включение синхронизации времени.
Конфигурация автоматических обновлений (Unattended Upgrades).

Версионирование.
x.y.z.a
Где,
x: Поколение. Это число меняется только при глобальном изменении способа реализации проэкта.
y: Количество добавленных новых функций или глобально переработанных старых.
z: Количество исправленных ошибок.
а: Количество косметических изменения кода, интерфейса, логики некоторых функций или команд. Эти изменения никак не влияют на функциональность и способ реализации выполняемых задач.
